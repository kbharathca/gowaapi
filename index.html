Endpoint Quick Reference

Text

POST /send/message

JSON: phone, message, is_forwarded, reply_message_id, duration.

Photo

POST /send/image

Form data: phone, image or image_url, caption, compress, view_once.

Video

POST /send/video

Form data: phone, video or video_url, caption, compress, view_once, gif_playback.

File

POST /send/file

Form data: phone, file, caption, is_forwarded, reply_message_id.

Every send request includes X-Device-Id. See API_DOCS.md for Cloudflare API examples.
